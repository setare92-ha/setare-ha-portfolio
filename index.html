Projects

  • Human Activity Recognition (HAR) Using Neural Networks

    Tags: logistic-regression, neural-networks, time-series-analysis, deep-learning

    Data: WISDM Lab

    Link: GitHub

    Description: This project focuses on developing a machine learning algorithm for HAR using data collected by the Wireless Sensor Data Mining (WISDM) Lab. The dataset includes labeled accelerometer data from 29 users as they perform six different daily activities: walking, jogging, ascending/descending stairs, sitting and standing. In the original work, 43 features were extracted from the time series data and three classification methods (decision trees, logistic regression and multilayer perceptron) were used to classify the activities. This work first reproduces the results of Kwapisz et al., 2010 for one of the conventional classification methods they studied. I will then show that we can improve the accuracy of the HAR algorithm by using a 1D convolutional neural networks.

  • Cardiovascular Event Detection

    Tags: random-forest, classification, hyper-parameter-tuning

    Data: UCI Heart Disease Dataset

    Link: GitHub

    Description: The objective of this project is to develop a machine learning model that can predict the susceptibility of patients to heart disease based on several factors that can be monitored during health checks. This can be used as an early detection mechanism to improve patient outcomes and/or to prevent the occurance of such disease by doing more frequenct health checks or through life style changes. To this end, we use the UCI Heart Disease Dataset. We explore using logistic regression, decision trees and random forest models. The analysis includes cross-validation, hyperparameter tuning and final evaluation of the models using appropriate metrics.

  • What Movies to Invest In?

    Tags: regression, data-cleaning-and-preprocessing, web-scraping

    Data: Source

    Link: GitHub

    Description: This project analyzes the profitability of movies by calculating their Return on Investment (ROI) using data from sources like Box Office Mojo, IMDB, Rotten Tomatoes, TMDB, and TheNumbers. By focusing on metrics such as movie budget, domestic and worldwide gross, genre, release month, and ratings, the project provides insights to stakeholders in the movie industry, including movie studios and streaming platforms. It aims to help stakeholders identify profitable movie trends and make informed decisions about movie production and acquisitions based on ROI and other influential factors.

  • Aviation Safety

    Tags: data-cleaning-and-preprocessing, eda, data-to-insights

    Data: Kaggle

    Link: GitHub

    Description: This project analyzes aviation crash data to identify aircrafts with the lowest safety risks, helping a company assess potential investments in airplanes for commercial and private enterprises.
